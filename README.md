## Overview

A multi‑page website for a hosting or tech services company called Grid Labs. The site describes services, pricing plans and contact information.

## Features

| ✔️ | Feature |
|---|---|
| ✅ | Services overview section |
| ✅ | Pricing tables for different plans |
| ✅ | Contact form and FAQ page |

## Installation

Follow these steps to get a local copy of the project up and running:

1. Clone the repository: `git clone https://github.com/Nischhalsubba/Grid-Labs-Hosting.git`.
2. Open `index.html` or run a local static server to view the site.

## Usage

Customize the text and images to match your company's branding. Deploy the site on a hosting platform.
